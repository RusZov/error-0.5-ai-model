# Теория «Ошибки 0.5»

## Место для загрузки вашей теории

Эта папка предназначена для хранения документов, описывающих научную теорию «ошибки 0.5».

### Как загрузить файл с теорией:

1. **Через GitHub UI**:
   - Перейдите в репозиторий
   - Нажмите "Add file" → "Upload files"
   - Перетащите ваш файл (PDF, DOCX, TXT, MD)
   - Commit changes

2. **Через командную строку**:
   ```bash
   cp /path/to/your/theory.pdf /workspace/error-0.5-ai-model/docs/
   git add docs/theory.pdf
   git commit -m "Add theory document"
   git push
   ```

3. **Через Issue**:
   - Создайте новый Issue
   - Прикрепите файл к сообщению
   - Мы обработаем и добавим в репозиторий

### Форматы файлов:
- ✅ PDF
- ✅ DOCX / DOC
- ✅ TXT
- ✅ MD (Markdown)
- ✅ LaTeX (.tex)
