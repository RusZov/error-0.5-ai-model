# error-0.5-ai-model
Реализация теории «ошибки 0.5» для анализа и управления вероятностным поведением языковых моделей. Расчёт коэффициента повторяемости (K_rep), энтропии (H), индекса нехватки данных (D_lack) и индекса полезной ошибки (Ω_0.5). Инструмент для экспериментов с повторяемостью ответов ИИ, фильтрацией шума и управлением температурой генерации.
